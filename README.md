# Crypto Exchange Frontend Concept (WordPress)

> A university project demonstrating complex frontend logic, user account management, and responsive design within WordPress.

## 📌 Project Overview

This project was an attempt to build a crypto/fiat exchange platform where users could:
- Fill out buy/sell order forms
- Make payments (simulated frontend)
- Manage their user account

The backend delivery of crypto/fiat was **not fully implemented** – this is a **frontend/demo concept** showcasing my ability to build complex forms, conditional math, and user systems.

## 🧠 What I Built & Learned

### Fully working features:
- **Responsive landing page** with modern layout
- **User account system** (registration, login, profile) – built with Ultimate Member plugin
- **Complex buy/sell order forms** with real‑time conditional math (e.g., price updates based on quantity)
- **Form validation** and submission flows

### What I learned:
- Handling dynamic form logic with JavaScript
- Extending WordPress functionality using plugins (Ultimate Member)
- The challenges of integrating payment/crypto delivery (backend complexity)

## 🛠️ Tech Stack

- WordPress
- Ultimate Member plugin (user management)
- JavaScript (vanilla, for form logic)
- HTML5 / CSS3
- UpdraftPlus (backup)
